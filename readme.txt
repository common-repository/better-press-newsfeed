A plugin to provide a dashboard widget for WP Tavern and Post Status.

== Description ==

A simple plugin to provide a dashboard widget for WP Tavern and Post Status news feeds. As the go-to news sites, some people want easy access to their news items. This gives you that.

== Installation ==

1. Upload `better-press-newsfeed` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Read the news

== Frequently Asked Questions ==

= What is this for? =

The news. WordPress news, to be specific.

= That it? =

Yep

= What if I only want one of them? =

Go to the "Screen Options" tab on the dashboard page and uncheck the box
